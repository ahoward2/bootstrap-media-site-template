About The Design Of This Website

I used a lot of Bootstrap4 for the components of the website, but I spent a lot of time customizing the overall design and overriding the Bootstrap CSS. Bootstrap handles the media queries for responsiveness when changing screen size but those can also be customized in CSS. I also used JQuery for some of the functionality of the site like this popup being activated by the button in the footer. Also, some of the icons are Bootstrap icons, and some are Fontawesome icons.

I like the cooler, darker colors in the design, and I changed the color balance of a lot of the images that I used in the carousels to make them more blue to match the design.
